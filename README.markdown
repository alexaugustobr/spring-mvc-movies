Spring Movies
=============

A sample application powered by:

* Spring MVC 3.1.1
* MongoDb 2.0.3
* Spring Security 3.1.0
* Thymeleaf 2.0.4

Usage
-----

The project includes a Gradle build script. 

The following command will create an eclipse project:

    gradle eclipse

to run the application inside tomcat:

    gradle runTomcatWar
    gradle stopTomcat
