Spring Movies
=============

A sample application powered by:

* Spring MVC 3.2
* MongoDb 2.2
* Spring Security 3.1.0
* Thymeleaf 2.0.13

Usage
-----

The project includes a Gradle build script. 

The following command will create the Eclipse project:

    gradle eclipse

to run the application inside tomcat:

    gradle runTomcatWar
    gradle stopTomcat
